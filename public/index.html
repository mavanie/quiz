This is a guide to how to use a github codespace to run a repository

First, you have to create a Github acount. Use a Username and a strong password that you will remember. Now you have a Github acount. Next, you need to make a repository. Click on the plus sign and hit new repository. Name the repository the name of the program. Then select public or private. Public means that anyone can see your repository. private means only people with permision can see. Both riquire permision to edit. Then press create repository.
